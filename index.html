Why Velas Staking?

The only requirement is to be the owner of more than 10,000 VLX, and you are done! Choose a pool and earn between 2-8% per year.

  • Passive way to increase your VLX balance while hodl.

    Staking allows you to participate in AIDPoS Velas consensus and generate rewards for it.

  • Easy, faster.

    1...2...3 Stake! Forget about tedious setups or having to leave your wallet open 24/7. Just choose a pool and start staking.

  • Decentralized. It's up to you.

    The process is done with smart contracts, you decide when to add or remove VLX from it.

Run your own Node without complications.

The <script> is generated from Velas wallet with a single click.

Save the script on your local machine or remote server and run it. Follow our step by step guide to node setup to know all the requirements.

EXAMPLE SCRIPT CODE 
<curl https://explorer.velas.com/cdn/install-velas.sh -L | bash">
  <mkdir -p ./data>
    <curl https://explorer.velas.com/cdn/spec.json > ./spec.json>
    <curl https://explorer.velas.com/cdn/config.toml?signer_acc=0xa9d900ae446b12d440b1ce99e71b710e24089032 > ./config.toml>
	<echo '8fb5a2e21f4fe825460e5ab152d20b74' >> ./password>
	<./velasblockchain --config ./config.toml account import signer_acc.json>
	<echo "./velasblockchain --config ./config.toml" >> ./start.sh>
  </chmod +x ./start.sh>
  <./start.sh>
